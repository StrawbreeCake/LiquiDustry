# LiquiDustry - a fluidic mod for Mindustry, the RTS game

The Java Mindustry mod template that works on Android and PC can be found [here](https://github.com/Anuken/MindustryJavaModTemplate).

## What's in this mod?

Well, right now it's just item-liquid transmografication. Turn any vanilla liquid into a gelled state, then turn it back again!

"Why would I do that?"

I'm glad you asked: DRONES!

## What's next?

This is my first foray into java and mod-making, so I can't promise quick results. :P
I hope to add proper liquid drones and mass drivers, as well as cool turrets and units! (Firefighting aircraft, anyone?) I also think that a unit-focused mod without turrets would be interesting.

Installation:
* Open "mods" in game
* Open "import from GitHub"
* Write: StrawbreeCake/LiquiDustry
